# Codeflow Session

Start a new development session using the Codeflow methodology.

## Workflow

1. **Discovery** - Research and gather information
2. **Second Opinion** - Get alternative perspectives (optional)
3. **Discussion** - Align on direction with the user
4. **Hearing** - Detailed Q&A to clarify requirements
5. **SDG** - Create spec and design documents
6. **Implementation** - Build with checklists
7. **Learning** - Record insights for future reference

## Usage

Begin by asking the user about the feature or task they want to work on, then follow the Codeflow phases.
